# 🌟 Oasis Infobyte Data Science Internship (OIBSIP)

## 📌 Internship Objective

This repository contains the projects completed as part of the Oasis Infobyte Data Science Internship Program.

The internship focused on applying Data Science and Machine Learning techniques to solve real-world problems through data analysis, visualization, predictive modeling, and dashboard development.


## 🛠 Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Plotly
* Scikit-learn
* NLTK
* Streamlit


## 🚀 Projects Completed

### 🌸 Task 1 – Iris Flower Classification

A machine learning project that classifies iris flowers into different species based on petal and sepal measurements.

**Key Skills:**

* Data Preprocessing
* Exploratory Data Analysis
* Classification
* Model Evaluation

**Model Used:** Random Forest Classifier

### 📊 Task 2 – Unemployment Analysis in India

An exploratory data analysis project that examines unemployment trends across India and studies the impact of COVID-19 on employment.

**Key Skills:**

* Data Cleaning
* Data Visualization
* Trend Analysis
* Correlation Analysis


### 📧 Task 4 – Email Spam Detection

A machine learning project that classifies emails as Spam or Not Spam using Natural Language Processing techniques.

**Key Skills:**

* Text Preprocessing
* TF-IDF Vectorization
* NLP
* Classification

**Model Used:** Support Vector Machine (SVM)

### 📈 Task 5 – Sales Prediction using Advertising Data

A machine learning project that predicts product sales based on advertising expenditure across TV, Radio, and Newspaper channels.

**Key Skills:**

* Data preprocessing
* Exploratory Data Analysis
* Feature Importance Analysis
* Predictive Modeling
* Model Comparison
* Business Insights

**Model Used:** RandomForestRegressor
