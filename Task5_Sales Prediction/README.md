# 📈 Sales Prediction using Machine Learning
## 🏆 Oasis Infobyte Internship Task

This project was completed as part of the **Oasis Infobyte Data Science Internship**.

The goal of this project is to build a machine learning model that can predict product sales based on advertising expenditure across TV, Radio, and Newspaper channels. Understanding the relationship between advertising expenditure and sales can help businesses plan their marketing efforts more efficiently and improve future sales performance.


## 📌 Objective

The objective of this project is to develop a sales prediction system using machine learning.

The model analyzes advertising budget inputs and predicts the expected units of product sold, helping businesses optimize their advertising strategy and forecast future revenue.


## 📂 Dataset

The dataset contains advertising expenditure and corresponding sales figures across different media channels.

**Dataset Summary:**

| Detail | Value |
|---|---|
| Total Records | 200 |
| Features | TV, Radio, Newspaper |
| Target Variable | Sales |
| Source | Advertising.csv |

•	Streamlit Deployment 


## 📊 Results

- Achieved an **R² score of 0.981**, meaning the model explains ~98% of the variance in sales.
- Successfully predicted sales values with an MAE of 0.620 and RMSE of 0.768.
- Demonstrated the effectiveness of Random Forest for regression tasks on advertising data.

## 🎯 Outcome

This project demonstrates how machine learning can be used to predict product sales from advertising expenditure. By combining exploratory data analysis, feature importance analysis, and a Random Forest Regressor, the system effectively forecasts sales and helps businesses optimize their advertising budget allocation.
