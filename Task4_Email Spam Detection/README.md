# 📧 Email Spam Detection using Machine Learning

## 🏆 Oasis Infobyte Internship Task

This project was completed as part of the Oasis Infobyte Data Science Internship.

The goal of this project is to build a machine learning model that can automatically classify emails as **Spam** or **Not Spam (Ham)**. Spam emails often include unwanted advertisements, fraudulent offers, or phishing attempts, making email filtering an important and practical use of machine learning.


## 📌 Objective

The objective of this project is to develop an email spam detection system using machine learning.

The model analyzes email content and predicts whether an email is spam or legitimate, helping users identify potentially harmful messages.


## 📂 Dataset

The dataset contains a collection of email messages labeled as:

* Spam
* Ham (Not Spam)


## 🛠 Tools & Technologies Used

* Python
* Pandas
* NumPy
* NLTK
* Scikit-learn
* Matplotlib
* Seaborn
* Streamlit
* Jupyter Notebook


## 🔍 Steps Performed

1. Loaded and explored the dataset.
2. Cleaned and preprocessed email text.
3. Converted text into numerical features using TF-IDF.
4. Trained machine learning models (Naive Bayes and SVM).
5. Evaluated model performance.
6. Compared model performance and selected SVM as the final model.
7. Built a Streamlit application for real-time spam detection.

## 📊 Results

* Achieved an accuracy of approximately 97.75%.
* Successfully classified emails into Spam and Not Spam categories.
* Demonstrated the effectiveness of TF-IDF and SVM for text classification tasks.


## 🎯 Outcome

This project demonstrates how machine learning can be used to automatically detect spam emails. By combining text preprocessing, TF-IDF vectorization, and an SVM classifier, the system effectively identifies unwanted messages and helps improve email security.
