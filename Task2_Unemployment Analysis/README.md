# 📊 Unemployment Analysis in India

## 🏆 Oasis Infobyte Internship Task

This project was completed as part of the Oasis Infobyte Data Science Internship.

The objective of this project is to analyze unemployment trends in India and understand how unemployment rates changed across regions and during the COVID-19 pandemic using data analysis and visualization techniques.



## 📌 Objective

The COVID-19 pandemic caused major disruptions in employment across the country. This project aims to:

•	Analyze unemployment trends in India

•	Compare unemployment rates across different regions

•	Study the impact of COVID-19 on unemployment

•	Identify patterns and relationships between employment-related factors



## 📂 Dataset

The dataset contains information such as:

•	Region

•	Date

•	Estimated Unemployment Rate (%)

•	Estimated Employed

•	Labour Participation Rate (%)



## 🛠 Technologies Used

•	Python

•	Pandas

•	NumPy

•	Matplotlib

•	Seaborn

•	Plotly

•	Jupyter Notebook



## 🔍 Steps Performed

### 1. Data Loading
- Imported the unemployment dataset.
- Examined dataset structure and dimensions.

### 2. Data Cleaning
- Removed unnecessary whitespaces from column names.
- Converted the Date column into datetime format.
- Renamed columns for better readability.
- Checked for missing values.

### 3. Exploratory Data Analysis (EDA)
- Analyzed overall unemployment trends over time.
- Examined unemployment distribution across regions.

### 4. Region-wise Analysis
- Calculated average unemployment rates for each region.
- Identified regions with the highest and lowest unemployment rates.

### 5. COVID-19 Impact Analysis
- Categorized data into:
  - Pre-COVID
  - During COVID
- Compared unemployment rates using boxplots.

### 6. Correlation Analysis
- Generated a correlation heatmap.
- Studied relationships between:
  - Unemployment Rate
  - Estimated Employed
  - Labour Participation Rate

### 7. Interactive Visualization
- Created interactive unemployment trend plots using Plotly.



## 📈 Key Findings

•	Tripura and Haryana showed comparatively higher unemployment rates.

•	Meghalaya recorded one of the lowest unemployment rates.

•	COVID-19 caused greater variation and instability in unemployment levels.

•	Employment-related factors showed noticeable correlations with unemployment trends.


## 🎯 Outcome

Successfully analyzed unemployment trends in India and evaluated the impact of the COVID-19 pandemic through data cleaning, exploratory data analysis, statistical visualization, and interactive dashboards.

This project analyzes unemployment trends across India, compares regional unemployment rates, and examines the impact of the COVID-19 pandemic on employment.
