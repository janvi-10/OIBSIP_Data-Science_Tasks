🌸 Iris Flower Classification (Machine Learning)

🏆 Oasis Infobyte Internship Task
This project was completed as part of the Oasis Infobyte Data Science Internship Program (OIBSIP).

📌 Project Overview
This project builds a Machine Learning classification model to predict the species of an Iris flower based on sepal and petal measurements.
The model is trained using the Iris dataset which contains 150 samples across 3 species:

-- Iris Setosa
-- Iris Versicolor
-- Iris Virginica

The model learns from 4 features — Sepal Length, Sepal Width, Petal Length, and Petal Width — and classifies the flower into the correct species with high accuracy.

🚀 Steps Performed 

Data loading and exploration using Pandas
Data visualization using Seaborn and Matplotlib
Train-test split (80% training, 20% testing)
Random Forest Classifier model training
Model evaluation using Accuracy, Precision, Recall and F1-Score
Confusion Matrix visualization
Feature Importance analysis
Prediction on new unseen flower measurements

📊 Model Performance
The Random Forest Classifier achieves high accuracy on the test dataset and performs well across all three Iris species categories.

💡 Key Learnings

Petal measurements are more useful than sepal measurements for classification
Random Forest handles multi-class classification natively
Iris dataset is clean and well-separated making it ideal for beginners in ML
Visualization plays a key role in understanding data before modeling
