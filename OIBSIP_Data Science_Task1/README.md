# 🌸 Iris Flower Classification (Machine Learning)

## 🏆 Oasis Infobyte Internship Task

This project was completed as part of the **Oasis Infobyte Data Science Internship Program (OIBSIP)**.

## 📌 Project Overview

This project builds a **Machine Learning classification model** to predict the species of an Iris flower based on sepal and petal measurements.

The model is trained using the **Iris Dataset**, which contains **150 samples** across three species:

* Iris Setosa
* Iris Versicolor
* Iris Virginica

The model learns from four features:

* Sepal Length
* Sepal Width
* Petal Length
* Petal Width

and classifies flowers into the correct species with high accuracy.

## 🚀 Steps Performed

* Data loading and exploration using Pandas
* Data visualization using Seaborn and Matplotlib
* Train-Test Split (80% Training, 20% Testing)
* Random Forest Classifier model training
* Model evaluation using Accuracy, Precision, Recall, and F1-Score
* Confusion Matrix visualization
* Feature Importance analysis
* Prediction on new unseen flower measurements

## 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib
* Seaborn

## 📊 Model Performance

The **Random Forest Classifier** achieves high accuracy on the test dataset and performs well across all three Iris species categories.

## 💡 Key Learnings

* Petal measurements are more informative than sepal measurements for classification.
* Random Forest can handle multi-class classification effectively.
* The Iris dataset is clean and well-separated, making it ideal for beginners in Machine Learning.
* Data visualization helps uncover patterns and understand feature importance before model training.
